# 서울교육공론화 클립북

> 서울교육 공론화 사업이 **이미 세상에 어떻게 발신·보도됐는지** 매체별로 모은 사전 자료 지형도

---

## 이 클립북이 뭐예요?

"**[프로젝트] 클립북**"은 새 프로젝트가 시작될 때 가장 먼저 만드는 자료 모음집입니다.

쉽게 말해 — **"이 주제, 이 사업, 이 기관이 지금까지 어떻게 발신돼 왔는지 한눈에 보여주는 사전 조사 결과물"**.

자료맵(어디에 무엇이 있다는 지도)이 아니라, **실제 자료(클립) 그 자체**를 모아둡니다.

## 왜 만드나요?

| 용도 | 설명 |
|------|------|
| 🎯 **사전 조사 단축** | 컨설턴트가 매번 처음부터 검색할 필요 없음 |
| 📋 **실행계획서 입력 자료** | 클립북 내용이 실행계획서 작성의 1차 입력으로 들어감 |
| 🔍 **클라이언트 대화 도구** | "작년에는 이런 보도가 있었습니다" 식의 근거 제시 |
| 📚 **백서·결과보고서 참고** | 사업 결과 보고 시 과거 맥락 인용 |

## 어떻게 정리돼 있나요?

**매체(channel)별로 묶고**, 각 매체 안에서 **연도순 클립**으로 정렬:

| ID | 매체 | 이 클립북에 들어있는 것 |
|----|------|----------------------|
| A | 교육청 공식 보도 (enews.sen.go.kr) | 서울교육청이 직접 낸 보도자료 |
| B | 외부 언론 보도 | 서울신문·뉴시스 등 미디어 클립 |
| C | 교육청 누리집 참여·의제 채널 | 서울교육 소통광장 등 시민 참여 채널 |
| D | 영상·카드뉴스·SNS | SEN TV·인스타그램 등 |
| E | 보고서·연구·웹진 | 서울미래교육 2030 보고서 등 |
