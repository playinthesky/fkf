## 산출물 구조

| 파일 | 단계 |
|------|------|
| `트랙별_정리.md` | 7트랙 카테고리화 (실행계획서의 골조) |
| `마스터_일정표.md` | 단계 정의 (시점은 협의 후 확정) |
| `협의필요_항목.md` | 클라이언트와 협의할 50+ 미정 항목 |
| `실행계획서_v0.x.y.html` | 최종 산출물 (마스터 = HTML) |
| `실행계획서_v0.x.y.hwpx` | 한컴 변환물 (별동수 또는 한컴 직접) |
| `실행계획서_v0.x.y.pptx` | PPT 변환물 (별동수 또는 Canva·Gamma 직접) |

## 버전 흐름

```
v0.1.0 (트랙·일정 골조 — 현재)
   ↓
v0.5.x (HTML 초안 — 분석+클립북 종합)
   ↓
v0.6.x (질문 답변 반영)
   ↓
v0.7.x ~ v0.9.x (클라이언트 협의 결과 반영)
   ↓
v1.0 (확정)
   ↓
.hwpx / .pptx 변환물
```
