## 2. 코리아스픽스 ↔ 글로벌리서치 역할 분담

### 2.1 코리아스픽스 (대표사)

| 과업 영역 | 세부 활동 |
|----------|----------|
| 총괄 관리·발주처 대응 | PM 총괄책임(이병덕 대표), 착수보고, 추진단 회의, 월별 추진 현황 보고 |
| 의제발굴 및 선정 | 의제 발굴 방법 설계, FGI 기획·운영(의제분석팀), 의제선정협의체 지원, 의사결정 모델 |
| 공론화 모델 설계 | 추진협의체 구성·운영 지원, 숙의 프로그램 설계(스픽스 방법론), 숙의자료집 제작, 온라인 토론장 기획 |
| 여론조사 (의제 연계) | 의제 연계 설계, 사전·사후 인식 변화 측정 문항, **토픽 모델링 기반 정성 분석** ⭐ STM |
| 숙의과정 운영 (오프라인) | 리딩 퍼실 팀, 토론회 기획·시나리오, 소그룹 토론(명목집단기법), 현장 진행 스태프, FT 팀장·갈등관리, 회의록·녹취, 안전대책·행사보험 |
| 온라인 공론장 | 소통광장 연계, 온라인 공론 결과 분석 |
| 결과 공표·결과문 | 전 과정 정성 분석, 정책제안문·권고안, 인포그래픽, 결과 공표 행사 기획 |
| 홍보 | 홍보 전략 총괄, 영상·인쇄물 기획·제작·배포, 단계별 보도자료 |
| 전담 조직 | 조직도·인력 프로필, 숙의컨설팅(이소연·정지혜), 기술관리(최대성), 운영지원(이소영·김재은) |

### 2.2 글로벌리서치 (참여사)

| 과업 영역 | 세부 활동 |
|----------|----------|
| 총괄 관리 협력 | 연구책임(이동열 수석부장), 실행계획 반영 협력 |
| 공론화 모델 설계 협력 | 시민참여단 모집 설계·표본 구성, 대표성 확보 방안 검토, 공론화 홈페이지 운영 지원 |
| 여론조사 (실시) | **여론조사 설계 (1,500명+)·운영**, 통계 분석·데이터 처리, 사전·사후 설문 운영 |
| 결과 공표·결과문 협력 | 여론조사 결과 분석보고서, 최종보고서 내 조사 통계 파트 |
| 홍보 협력 | 홍보물 제작 실무, 영상 촬영·편집, **공론화 백서 구성·제작 총괄** ⭐ |

→ 사장님 메모 "백서는 글로벌팀" = 글로벌리서치(주) (우리 부서가 아니라 **외부 참여사**) 백서 담당이 명확함.

## 4. 발주처(서울시교육청)의 협력 역할

컨소시엄 구성원은 아니나 사업 전 과정 협력:

| 역할 | 내용 |
|------|------|
| 소통광장 플랫폼 제공 | 공론화 메뉴, '시민제안'·'정책설문' 채널, 홍보·경과·결과 게시 공간 |
| 협의체 운영 지원 | 의제선정협의체 운영 주관 (1~2회), 추진협의체 위원 선정 관여·승인 |
| 전 과정 승인·모니터링 | 세부 시행계획 검토·승인, 월별 추진 현황 수령, 이슈 보고 수령 |
| 홍보 협력 | 교육청 공식 채널을 통한 공론화 사업 홍보, 유관 부서·학교 네트워크 연계 |
| 결과 수령·정책 반영 | 최종 결과보고서·정책제안문 검수, 해당 부서 정책 권고 전달, 2027 정책 반영 |

## 5. 운영 원칙

### 5.1 보고 라인
- 코리아스픽스 PM → 서울시교육청 담당자: **매주 진행 현황 보고, 수시 특이사항**
- 월말 기준 과업 추진 현황 **익월 5일까지 정기 제출**
- 발주처 요청 시 수시 보고

### 5.2 컨소시엄 내부 조율
- **주간 진행 상황 공유 회의** (코리아스픽스 ↔ 글로벌리서치)
- 여론조사 문항·결과는 **양사 사전 검토·합의 후 확정**
- 성과품 제출 전 **양사 교차 검토**

### 5.3 과업 이행 준수
- 모든 과업 사전 협의 후 진행
- 참여인력 교체 사전 승인 필수
- **개인정보 수령 즉시 암호화·보안 유지**
- **행사보험 가입 (피보험인: 서울시교육청, 보험증권 원본 제출)**
