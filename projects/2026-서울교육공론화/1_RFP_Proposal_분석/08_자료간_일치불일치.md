## 2. ⚠️ 불일치 (확인·정정 필요)

### 2.1 🚨 수주 형태 — **사장님 메모 정정 필요**

| 자료 | 명시 |
|------|------|
| 사장님 메모 (구두) | "단독수주" |
| 업무분담초안 | **컨소시엄**: 코리아스픽스(주관) + 글로벌리서치(참여) |
| RFP | 공동수급체 허용 (구성원 1개·최소 5%) |

→ **실제는 컨소시엄.** 사장님이 처음에 "단독수주"라고 한 것은 잘못된 정보. 사장님 메모 정정 + 모든 후속 산출물에 컨소시엄으로 반영.

### 2.2 🚨 추진위 회의 주기

| 자료 | 명시 |
|------|------|
| 스픽스 마스터 정의서 (표준) | 월 1회 정기, 필요시 수시 |
| RFP·과업지시서 | "3~4회" (총 횟수만, 주기 없음) |
| 사장님 메모 | "사안별 수시" (정기 주기 없음) |
| 2025 실제 | 약 2~4주 간격, 총 4차 (10/13, 10/20, 11/17, 12월) |

→ **올해 서울교육공론화 = "사안별 수시"** (사장님 결정). 마스터 정의서의 "월 1회"는 일반 숙의 표준이므로 본 프로젝트에는 적용 ❌.
