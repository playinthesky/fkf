## 분석 원칙

1. **사실만 기록**: 자료에 적힌 그대로 (해석·추측 ❌)
2. **출처 명시**: 모든 정보에 자료명·페이지 표기
3. **갭 명시**: 자료에 없는 항목은 "자료에 없음" 표기
4. **08번이 핵심**: 자료 간 불일치를 찾아내는 것이 분석의 가장 큰 가치
