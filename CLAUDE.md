# AX 프로젝트 컨텍스트

## 핵심 개념

### 우리의 솔루션: 숙의형 데이터 분석
- 정성적 분석 + 구조적 토픽 모델링 (STM)
- **모든 프로젝트에 내장**되는 핵심 기술

### AX (AI + Human)
- 모든 프로젝트는 Human + AI 협업 방식
- AX 컨설팅 = 우리가 제공하는 서비스

### 프로젝트 구조
```
[고객 프로젝트]
├─ 솔루션: 숙의형 데이터 분석 (STM)  ← 항상 포함
├─ 방식: AX (Human + AI)             ← 항상 포함
└─ 의제: 참여예산 / 도시계획 / ...    ← 프로젝트별 다름
```

## 코드체계 요약

### 대분류 (A~E)
- **A** 전략/기획
- **B** 개발/구축
- **C** 운영/관리
- **D** 교육/전환
- **E** 품질/감리

### 인력등급 (S~D)
| 등급 | 경력 | M/M 단가(천원) |
|------|------|---------------|
| S | 20년+ | 18,000 |
| A | 12년+ | 14,000 |
| B | 6년+ | 10,000 |
| C | 3년+ | 7,000 |
| D | 3년- | 4,500 |

## 작업 규칙

1. 새 프로젝트 산출내역서 → `templates/산출내역서_템플릿_v1.0.0.md` 복사 후 수정
2. 새 숙의 프로젝트 → `templates/숙의프로젝트_템플릿_v1.0.0.md` 복사 후 수정
3. 모든 견적에 "숙의형 데이터 분석" 항목 포함
4. AX 방식 명시 (Human + AI 협업)
5. 정부 보고용 파워포인트는 4:3 규격(10" × 7.5")으로 만든다
