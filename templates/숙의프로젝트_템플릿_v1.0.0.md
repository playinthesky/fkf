# AX 숙의(熟議) 프로젝트 템플릿
> Version 1.0.0 | AX 표준화 시리즈

---

## 사용 안내

이 템플릿은 **시민참여형 숙의(Deliberation) 프로젝트**를 기획·운영하기 위한 범용 양식입니다.

### 적용 분야
- 공공정책 (예산, 조례, 도시계획)
- 기술·윤리 (AI 윤리, 데이터 거버넌스)
- 지역현안 (환경, 교통, 복지)
- 기업 내 의사결정 (전략, 조직문화)
