## 공통 스타일 키워드
```
Color palette: deep navy blue (#1E3A5F), teal (#2A9D8F), warm gold (#E9C46A)
Style: professional, sophisticated, subtle depth, corporate premium
Lighting: soft ambient light from top-left
Background: transparent PNG or light gray (#F5F6F7)
```

## 1. 헤더 바 (Header Bar)

### 1-1. 기본 헤더
```
Create a horizontal header bar for presentation slides.

Specifications:
- Width: 1200px, Height: 80px
- Base color: deep navy blue (#1E3A5F)
- Subtle inner shadow for depth (not flat)
- Right side accent: small geometric element (abstract triangle or parallelogram) in teal (#2A9D8F), slightly overlapping the edge
- Very subtle gradient: navy to slightly lighter navy (barely noticeable)
- Soft drop shadow beneath (opacity 15%, blur 8px, offset 4px down)
- Clean, minimal, premium corporate feel
- Transparent background (PNG)

Style reference: high-end consulting firm presentation, McKinsey-level sophistication
```

### 1-2. 섹션 구분 헤더
```
Create a section divider header bar for slides.

Specifications:
- Width: 1200px, Height: 60px
- Layered design: main bar in navy (#1E3A5F) with a thin accent strip (3px) in gold (#E9C46A) running along the bottom edge
- Right corner: subtle angular cutout or chamfer with teal (#2A9D8F) fill peeking through
- Depth effect: the bar appears to have thickness, like a 3D ribbon viewed from slight angle
- Soft shadow suggesting the bar floats 2mm above the surface
- Premium, understated, professional

Transparent PNG background.
```

## 2. 구분선 (Divider Lines)

### 2-1. 다층 구분선
```
Design a horizontal divider line with layered depth effect.

Specifications:
- Width: 1000px, Height: 20px
- NOT a simple flat line
- 3 subtle layers stacked:
  - Bottom layer: soft shadow/blur (gray, 30% opacity)
  - Middle layer: thin dark navy line (2px)
  - Top layer: very thin teal accent line (1px) offset slightly
- The layers create a sense of depth like looking at edges of stacked paper
- Elegant, minimal, sophisticated
- Transparent background

Think: the refined edge of premium stationery, not a basic horizontal rule.
```

### 2-2. 코너 장식 구분선
```
Create an L-shaped corner divider element.

Specifications:
- Size: 150px x 150px
- Layered corner bracket design
- Multiple thin lines (2-3) in navy and teal, slightly offset from each other
- Creates depth through layering, not through heavy shadows
- Small circular or diamond accent at the corner intersection
- Use for slide corners or section markers
- Transparent PNG

Style: architectural blueprint meets premium annual report
```

- Shadow: realistic soft shadow suggesting box floats 4-5mm above surface

- Top edge: thin accent line in teal (#2A9D8F), 3px
- Rounded corners: 8px radius
- Interior is empty (for text overlay in PPT)
- Transparent background outside the box

Premium feel, like a physical card floating.

- Main box: white to very light gray subtle gradient
- The accent bar casts a tiny shadow onto the main box area
- Overall box has floating shadow beneath
- Rounded corners on right side only (8px)
- Clean, professional, suggests importance

Transparent PNG background.

### 4-1. 넘버 뱃지 세트
```
Create a set of numbered badges (1 through 5) with 3D depth.

Specifications:
- Each badge: 60px x 60px
- Shape: rounded square (12px radius)
- 3D effect: isometric-style, showing top and one side face
- Top face: navy (#1E3A5F) with the number in white
- Side face: darker navy (shadow side)
- Subtle highlight on top edge suggesting light source
- Numbers in clean sans-serif font
- Each badge has small drop shadow

Style: like the stacked squares in the infographic reference image, but smaller and as individual badges.

Create all 5 in one image, arranged horizontally with spacing.
Transparent background.
```

### 4-2. 체크마크/완료 아이콘

Design a checkmark icon with dimensional quality.

- Circle has subtle 3D spherical shading (not flat)
- White checkmark with very subtle thickness/dimension
- Small highlight reflection on the circle (premium glass-like quality)
- Soft drop shadow

One icon, transparent PNG background.
```

  2. Diamond shape with beveled edges (teal)
  3. Small arrow/chevron with 3D thickness (gold)

### 7-1. AX 테마 아이콘 (숙의/협업)
```
Create a set of 6 icons representing deliberation and collaboration themes.

Specifications:
- Each icon: 80px x 80px
- Style: dimensional, with subtle 3D quality (not flat line icons)
- Color: navy (#1E3A5F) with teal (#2A9D8F) accents
- Icons needed:
  1. People/group discussion (3 figures talking)
  2. Data/chart (bar graph with upward trend)
  3. Lightbulb/idea (insight moment)
  4. Handshake/agreement (collaboration)
  5. Document/report (professional paper)

## 활용 팁

3. **조합**: 요소들을 PPT에서 조합하여 사용
4. **색상 조정**: PPT에서 색조 조정 가능
