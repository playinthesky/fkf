# AX 프로젝트 산출내역서 템플릿
> Version 1.0.0 | AX 표준화 시리즈 | 수의계약 트랙 마스터

---

## 사용 안내

이 템플릿은 AX 표준 코드체계를 기반으로 **어떤 프로젝트에도 적용 가능한** 산출내역서 양식입니다.

### 적용 방법
1. `[프로젝트명]` 부분을 실제 프로젝트명으로 교체
2. 해당 프로젝트에 필요한 코드 항목만 선택
3. 수량, 등급, 배율을 프로젝트에 맞게 조정
4. 자동 계산 수식에 따라 금액 산출

### C.3 표준 Phase 구성 (참고)
```
[프로젝트명]
├─ Phase 1: 착수 및 분석 (전체의 10~15%)
│  ├─ 착수 보고
│  ├─ 현황 분석 (A01)
│  └─ 요구사항 정의 (A02)
├─ Phase 2: 설계 (전체의 10~20%)
│  ├─ 아키텍처 설계 (A03)
│  ├─ 데이터 모델 설계
│  └─ UI/UX 설계
├─ Phase 3: 구축 (전체의 50~60%)
│  ├─ 인프라 구축 (B07)
│  ├─ 개발 (B01~B06)
│  └─ 데이터 구축
├─ Phase 4: 테스트 (전체의 10~15%)
│  ├─ 통합 테스트 (E02)
│  ├─ 인수 테스트
│  └─ 보안 점검 (E04)
└─ Phase 5: 종료 (전체의 5~10%)
   ├─ 교육 (D01, D02)
   ├─ 인수인계
   └─ 종료 보고
```
