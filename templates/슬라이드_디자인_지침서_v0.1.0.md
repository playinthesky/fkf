```

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  굵은선 (강조)

──────────●──────────●──────────  포인트선 (단계)
```

```
                │
```

## 5. 말머리 기호 (Bullets)

### 5.1 기본 불릿
```
Level 1:  ●  채움 원 (AX Teal)
Level 2:  ○  빈 원 (Mid Gray)
Level 3:  ─  대시 (Light Gray)
Level 4:  ·  점 (Light Gray)
```

### 5.2 숫자 불릿
```
Level 1:  ①②③  원숫자 (AX Blue 배경, 흰색 글씨)
Level 2:  1. 2. 3.  점숫자 (Dark Gray)
Level 3:  a) b) c)  알파벳 (Mid Gray)
```

### 5.3 아이콘 불릿
```
✓  체크 (완료, Success)
→  화살표 (진행, 다음)
★  별표 (중요, 강조)
◆  다이아몬드 (핵심)
▶  삼각형 (액션, 실행)
```

### 5.4 상태 불릿
```
🟢  완료 / 정상
🟡  진행중 / 주의
🔴  미완료 / 문제
⚪  미정 / 해당없음
```

## 8. 차트 스타일

### 8.1 기본 원칙
- 3D 효과 사용 금지
- 그라데이션 최소화
- 데이터 레이블 명확히
- 범례는 차트 내부 또는 우측

### 8.2 색상 순서
```
1번째 데이터: AX Blue (#1E3A5F)
2번째 데이터: AX Teal (#2A9D8F)
3번째 데이터: AX Gold (#E9C46A)
4번째 데이터: #E17055 (Coral)
5번째 데이터: #74B9FF (Sky)
6번째 이상: 위 색상의 50% 틴트
```

### 8.3 그리드/축
- 축선: Mid Gray 1pt
- 그리드: Light Gray 0.5pt 점선
- 축 레이블: 14pt Mid Gray

## 10. 금지 사항

### 10.1 하지 말 것
- ❌ 워드아트, 그림자 텍스트
- ❌ 3D 차트, 3D 도형
- ❌ 클립아트 스타일 이미지
- ❌ 과도한 애니메이션
- ❌ 지정 색상 외 사용
- ❌ 12pt 미만 텍스트
- ❌ 저해상도 이미지 (150dpi 미만)

### 10.2 주의할 것
- ⚠️ 한 슬라이드에 메시지 1개
- ⚠️ 텍스트 최소화, 시각화 우선
- ⚠️ 여백 충분히 확보
- ⚠️ 일관된 정렬 유지
