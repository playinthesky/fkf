## 이 문서는

코리아스픽스의 **숙의·공론 컨설팅 표준 업무 플로우**입니다.

- 신규 컨설턴트 교육용
- 프로젝트 진행 체크리스트
- AI 학습 기반 (매일 업데이트)

---

## 5단계 라이프사이클

```
┌─────────┐   ┌─────────┐   ┌─────────┐   ┌─────────┐   ┌─────────┐
│ 1.PLAN  │ → │ 2.BUILD │ → │ 3.LAUNCH│ → │ 4.RUN   │ → │ 5.EVOLVE│
│  수주    │   │  설계    │   │  착수   │   │  실행   │   │  환류   │
└─────────┘   └─────────┘   └─────────┘   └─────────┘   └─────────┘
```

| 단계 | 이름 | 기간 | 핵심 질문 |
|------|------|------|----------|
| 1 | PLAN (수주) | 1~4주 | 누가, 왜, 얼마에? |
| 2 | BUILD (설계) | 2~4주 | 어떻게 할 것인가? |
| 3 | LAUNCH (착수) | 1~2주 | 준비 완료인가? |
| 4 | RUN (실행) | 4~12주 | 제대로 진행되나? |
| 5 | EVOLVE (환류) | 2~4주 | 무엇을 배웠나? |
