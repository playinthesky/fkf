# 3단계: LAUNCH (착수)

> 실행 직전 마지막 점검. 모든 것이 준비되었는가.

---

## 목표

- 참여자 확정·안내
- 퍼실리테이터 교육
- 장소·장비 최종 점검

---

## 기간

1~2주

---

## 핵심 질문

| 질문 | 왜 중요한가 |
|------|------------|
| 참여자 확정됐나? | 노쇼 방지 |
| 퍼실리테이터 준비됐나? | 진행 품질 |
| 장소·장비 문제없나? | 현장 사고 방지 |
| 발주처와 소통됐나? | 기대치 정렬 |
| 비상 계획 있나? | 돌발 대응 |

---

## 세부 단계

### 3.1 참여자 확정
- 모집 결과 확인
- 참여 확정 연락
- 사전 안내 발송

**체크리스트:**
- [ ] 모집 마감·인원 확인
- [ ] 참여 확정 문자/이메일
- [ ] 사전 안내문 발송 (일시/장소/준비물)
- [ ] 참여 수당 안내
- [ ] 노쇼 대비 대기자 확보

### 3.2 퍼실리테이터 교육
- 가이드 숙지
- 리허설 진행
- 역할 분담 확정

**체크리스트:**
- [ ] 퍼실리테이터 가이드 배포
- [ ] 사전 미팅 진행
- [ ] 리허설 (1회 이상)
- [ ] 역할별 담당 확정
- [ ] 비상 연락망 공유

### 3.3 장소·장비 점검
- 현장 답사
- 장비 테스트
- 동선 확인

**체크리스트:**
- [ ] 장소 답사 (최소 1회)
- [ ] 테이블·의자 배치 확인
- [ ] 마이크·스피커 테스트
- [ ] 프로젝터·스크린 테스트
- [ ] 인터넷/와이파이 확인
- [ ] 주차·접근성 확인

### 3.4 자료 최종 준비
- 인쇄물 준비
- 키트 제작
- 시스템 점검

**체크리스트:**
- [ ] 학습자료 인쇄
- [ ] 참여자 키트 제작 (명찰/볼펜/메모지)
- [ ] 투표용지/스티커 준비
- [ ] AI BDL 로그인 테스트
- [ ] 백업 자료 USB 준비

### 3.5 킥오프 미팅
- 발주처 최종 확인
- 내부 팀 브리핑
- D-1 점검

**체크리스트:**
- [ ] 발주처 킥오프 미팅
- [ ] 최종 일정·역할 확인
- [ ] 비상 연락처 공유
- [ ] D-1 전체 점검 완료

## 역할

| 역할 | 담당 업무 |
|------|----------|
| 담당 컨설턴트 | 전체 조율, 발주처 소통 |
| 퍼실리테이터 | 리허설 참여, 가이드 숙지 |
| 운영팀 | 장소·장비·키트 준비 |
| 대표 | 킥오프 참석 (필요시) |
| Claude | 안내문 초안, 체크리스트 점검 |

---

## 주의사항

### ⚠️ 흔한 실수
1. **노쇼 미대비** — 대기자 없음
2. **장비 미테스트** — 당일 고장
3. **리허설 생략** — 퍼실리테이터 혼란
4. **동선 미확인** — 참여자 길 잃음

### ✅ 좋은 습관
1. 참여자에게 D-3, D-1 리마인드
2. 장비는 반드시 현장 테스트
3. 백업 장비 준비 (마이크, 노트북)
4. 비상 시나리오 3가지 준비

---

## 비상 대응 시나리오

| 상황 | 대응 |
|------|------|
| 참여자 노쇼 다수 | 대기자 투입 / 소그룹 재편성 |
| 장비 고장 | 백업 장비 / 아날로그 전환 |
| 퍼실리테이터 불참 | 대체 퍼실리테이터 / 역할 재배치 |
| 발주처 일정 변경 | 신속 공지 / 보상 안내 |
