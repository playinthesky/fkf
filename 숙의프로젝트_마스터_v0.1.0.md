# 숙의(熟議) 프로젝트 마스터 정의서
> Version 0.1.0 | 2026-05-11 | AX 표준화 시리즈

---

## 1. 개요

### 1.1 정의
**숙의(熟議, Deliberation)**란 다양한 이해관계자가 충분한 정보를 바탕으로 심층적 토론을 거쳐 합의에 도달하는 민주적 의사결정 방식이다.

### 1.2 AX 숙의 프로젝트 목표
- 시민·전문가·행정이 함께 공공 의제를 심의
- AI 기반 퍼실리테이션으로 숙의 품질 향상
- 표준화된 프로세스로 재현 가능한 결과 도출

### 1.3 적용 분야
| 분야 | 예시 |
|------|------|
| 공공정책 | 예산 배분, 조례 제정, 도시계획 |
| 기술·윤리 | AI 윤리, 데이터 거버넌스 |
| 지역현안 | 환경, 교통, 복지 |

## 2. 위원회 구조

### 2.1 거버넌스 체계

```
┌─────────────────────────────────────────────────────────────┐
│                    총괄위원회 (Steering)                      │
│         의장 1인 + 분과장 + 외부전문가 (5~7인)                  │
└───────────────────────────┬─────────────────────────────────┘
                            │
        ┌───────────────────┼───────────────────┐
        ▼                   ▼                   ▼
┌───────────────┐   ┌───────────────┐   ┌───────────────┐
│  기획분과      │   │  운영분과      │   │  평가분과      │
│ (Planning)    │   │ (Operation)   │   │ (Evaluation)  │
└───────────────┘   └───────────────┘   └───────────────┘
        │                   │                   │
        ▼                   ▼                   ▼
┌───────────────┐   ┌───────────────┐   ┌───────────────┐
│ 시민배심원단   │   │ 전문가자문단   │   │ 모니터링단     │
│ (Citizen Jury)│   │ (Expert Panel)│   │ (Monitoring)  │
└───────────────┘   └───────────────┘   └───────────────┘
```

### 2.2 위원회별 구성 및 역할

#### 2.2.1 총괄위원회 (Steering Committee)
| 역할 | 인원 | 자격 | 책임 |
|------|------|------|------|
| 의장 | 1명 | 관련 분야 권위자 | 최종 의사결정, 대외 대표 |
| 분과장 | 3명 | 각 분과 대표 | 분과 운영 총괄 |
| 외부전문가 | 2~3명 | 숙의민주주의·해당분야 전문가 | 자문 및 견제 |

- **회의 주기**: 월 1회 정기, 필요시 수시
- **의결 정족수**: 재적 과반 출석, 출석 과반 찬성

#### 2.2.2 기획분과 (Planning Division)
| 역할 | 인원 | 주요 업무 |
|------|------|----------|
| 분과장 | 1명 | 기획 총괄 |
| 의제설계 전문가 | 2명 | 숙의 의제 구조화 |
| 자료조사 연구원 | 2명 | 배경자료·팩트체크 |
| AI/데이터 담당 | 1명 | AX 도구 연동 |

**산출물**: 숙의 의제서, 학습자료 패키지, 일정표

#### 2.2.3 운영분과 (Operation Division)
| 역할 | 인원 | 주요 업무 |
|------|------|----------|
| 분과장 | 1명 | 운영 총괄 |
| 퍼실리테이터 | 3~5명 | 토론 진행·중립성 유지 |
| 기록관 | 2명 | 논의 기록·녹취록 정리 |
| 행정지원 | 2명 | 장소·예산·물류 |

**산출물**: 회의록, 참여자 피드백, 운영 리포트

#### 2.2.4 평가분과 (Evaluation Division)
| 역할 | 인원 | 주요 업무 |
|------|------|----------|
| 분과장 | 1명 | 평가 총괄 |
| 프로세스 평가자 | 2명 | 절차 적정성 검증 |
| 결과 분석가 | 2명 | 합의안 품질 평가 |
| 외부 감사 | 1명 | 독립적 검증 |

**산출물**: 평가보고서, 개선권고안, 인증서

#### 2.2.5 시민배심원단 (Citizen Jury)
| 구성 | 인원 | 선발 방식 |
|------|------|----------|
| 일반시민 | 15~30명 | 무작위 추첨 (인구통계 층화) |
| 이해관계자 | 5~10명 | 공모 + 추천 |

**특징**:
- 성별·연령·지역 균형 고려
- 사전 학습 필수 (2~4시간)
- 숙의 수당 지급 (1일 10~15만원)

#### 2.2.6 전문가자문단 (Expert Advisory Panel)
| 분야 | 인원 | 역할 |
|------|------|------|
| 해당 분야 전문가 | 3~5명 | 기술적 자문 |
| 법률 전문가 | 1명 | 법적 검토 |
| 윤리 전문가 | 1명 | 윤리적 검토 |

**운영 원칙**: 발언은 자문에 한정, 의결권 없음

#### 2.2.7 모니터링단 (Monitoring Group)
| 구성 | 인원 | 역할 |
|------|------|------|
| 언론인 | 1~2명 | 투명성 감시 |
| 시민단체 | 1~2명 | 공정성 모니터링 |
| 학계 | 1명 | 학술적 관찰 |

## 3. 표준 프로세스 (5단계)

### 3.1 프로세스 개요

```
[1. 준비]──▶[2. 학습]──▶[3. 숙의]──▶[4. 합의]──▶[5. 환류]
 2~4주       1~2주       2~4주       1주        지속
```

### 3.2 단계별 상세

#### Stage 1: 준비 (Preparation) — 2~4주
| 활동 | 담당 | 산출물 |
|------|------|--------|
| 의제 확정 | 기획분과 | 숙의 의제서 |
| 위원회 구성 | 총괄위 | 위촉장 |
| 참여자 모집 | 운영분과 | 참여자 명단 |
| 자료 준비 | 기획분과 | 학습자료 패키지 |
| AX 도구 세팅 | AI담당 | 시스템 구축 완료 |

**체크리스트**:
- [ ] 의제 범위 명확화
- [ ] 예산 확보
- [ ] 장소 섭외
- [ ] 참여자 다양성 검증

#### Stage 2: 학습 (Learning) — 1~2주
| 활동 | 담당 | 산출물 |
|------|------|--------|
| 사전학습 자료 배포 | 기획분과 | 배포 완료 확인서 |
| 오리엔테이션 | 운영분과 | 참석자 서명부 |
| 전문가 브리핑 | 자문단 | 브리핑 녹화본 |
| Q&A 세션 | 퍼실리테이터 | Q&A 정리본 |

**학습 내용**:
1. 의제 배경 및 현황
2. 주요 쟁점 및 선택지
3. 각 선택지의 장단점
4. 숙의 프로세스 안내

#### Stage 3: 숙의 (Deliberation) — 2~4주
| 회차 | 형식 | 목표 |
|------|------|------|
| 1회차 | 소그룹 토론 | 쟁점 파악 |
| 2회차 | 전문가 패널 | 심화 이해 |
| 3회차 | 교차 토론 | 관점 교환 |
| 4회차 | 전체 토론 | 논점 수렴 |

**퍼실리테이션 원칙**:
- 중립성 유지
- 동등한 발언 기회
- 근거 기반 토론 유도
- AI 실시간 요약 지원

#### Stage 4: 합의 (Consensus) — 1주
| 활동 | 담당 | 산출물 |
|------|------|--------|
| 합의안 초안 작성 | 기획분과 | 초안 문서 |
| 투표/조율 | 시민배심원단 | 투표 결과 |
| 최종안 확정 | 총괄위 | 최종 합의문 |
| 서명식 | 전원 | 서명 합의문 |

**합의 방식**:
- 만장일치 추구 (1차)
- 숙의투표 (2차, 2/3 이상)
- 소수의견 부기

#### Stage 5: 환류 (Follow-up) — 지속
| 활동 | 담당 | 산출물 |
|------|------|--------|
| 결과 공개 | 운영분과 | 공개 보고서 |
| 이행 모니터링 | 모니터링단 | 이행점검표 |
| 평가 보고 | 평가분과 | 평가보고서 |
| 개선 반영 | 총괄위 | 개선계획서 |

## 6. 품질 기준

### 6.1 숙의 품질 지표 (DQI: Deliberation Quality Index)
| 지표 | 측정 방법 | 목표치 |
|------|----------|--------|
| 참여율 | 출석/등록 | ≥90% |
| 발언 균형 | 지니계수 | ≤0.3 |
| 근거 활용 | 팩트 인용률 | ≥60% |
| 상호 존중 | 부정 표현 비율 | ≤5% |
| 합의 수준 | 최종 동의율 | ≥70% |

### 6.2 인증 체계
- **Bronze**: 기본 프로세스 준수
- **Silver**: DQI 70점 이상
- **Gold**: DQI 85점 이상 + 이행 검증
- **Platinum**: Gold + 사회적 임팩트 검증
