## 한 줄 요약

**상담형 봇**: 한 방에 PPT 뱉어내는 게 아니라, 자료를 받고 대화로 빈 곳을 채워가며 같이 만드는 봇.

```
직원 ── /실행계획서 ──▶ 봇이 스레드 열고 자료 요청
       (파일 업로드)        ↓
                          자료 분석 + 빠진 정보 질문
                          ↓
                          드래프트 → 검토 → 수정 반복
                          ↓ (확정 시)
                          PPT + HWPX 생성 → Slack 업로드 + GitHub 커밋
```
